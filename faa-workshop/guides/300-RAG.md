# Add a knowledge database agent (RAG)

1. Kill the Solace Agent Mesh instance if its already running
    ```
    CTR + C
    ```

1. Execute the following from your `sam` dir
    ```
    cp ../solution/configs/agents/sop-agent.yaml configs/agents/sop-agent.yaml
    sam run
    ```
